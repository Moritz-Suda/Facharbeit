# Facharbeit
Dies ist das GitHub repository für meine abi Facharbeit

# Setup
- Python 3.14
- Pygame

# Usage
- `python -m venv .env`
- `sourc .env/bin/activate`
- `pip install pygame`
- `python3.14 ./src/main.py`

# Inhalte
- `main.py`: Hauptprogramm
- `grid.py`: Grid-Klasse
- `node.py`: Node-Klasse
- `pathfinding.py`: Pfadfindungsalgorithmen
- `visualizer.py`: Visualisierung
